# E-Commerce Sales Analytics Dashboard | Power BI

An interactive **Power BI dashboard** developed to analyze E-Commerce sales performance and provide business insights through dynamic visualizations and KPIs.

---

##  Project Overview

This project focuses on analyzing E-Commerce sales data to help businesses understand:

- Sales Performance
- Profit Analysis
- Customer Insights
- Product Performance
- Brand Performance
- Regional Sales
- Monthly Sales Trends

The dashboard is fully interactive using slicers and filters.
